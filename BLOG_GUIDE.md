# 전문적이고 시각적인 블로그 포스트 생성 가이드 (2026-04-08 업데이트)

본 가이드는 파트너(truplaya2)의 요청에 따라 모든 블로그 포스트 생성 시 반드시 준수해야 하는 지침입니다.

## 1. 기본 설정
- **최종 산출물**: HTML 본문(인라인 스타일, 코드블럭) + 부가 정보(텍스트)
- **컬러 테마**: 클래식 블루 (#1a73e8)
- **분량**: 한글 기준 공백 포함 2,500 ~ 3,000자
- **대상 독자**: 정보 제공형 콘텐츠에 관심 있는 일반인
- **톤앤매너**: 친근한 대화체(클로이 페르소나), 1인칭 시점, 생동감 있는 감정 표현

## 2. HTML 구조 및 스타일 지침
- **Wrapper**: `<div style="font-family: 'Noto Sans KR', sans-serif; line-height: 1.6; max-width: 800px; margin: 0 auto; font-size: 16px; box-sizing: border-box;">`
- **메타 설명 박스**: 회색 배경, 이탤릭체, 상단 배치
- **H2 제목**: 22px, 클래식 블루 색상, 하단 보더(2px solid #eaeaea)
- **본문 P**: 하단 여백 15px, 짧은 단락 구성
- **[핵심] 여백 관리**: **두 줄 이상의 연속된 줄바꿈(빈 줄) 및 불필요한 공백을 엄격히 금지**한다. 섹션 간에도 과도한 빈 줄을 넣지 않고 컴팩트하게 구성한다.
- **하이라이트**: `<span style="background-color: #fffde7; padding: 2px 4px; border-radius: 3px;">`
- **기타 요소**: 테이블(Table), 목록(UL/OL), 예시 박스 활용 권장
- **요약 섹션**: 끝부분에 📝 이모티콘과 함께 핵심 정리 포함
- **JSON-LD**: 본문 내에 스키마 데이터 포함

## 3. 콘텐츠 품질 체크리스트
- [ ] **여백**: 두 줄 이상의 빈 줄이 없는가?
- [ ] **메타 설명**: 핵심을 요약하는가?
- [ ] **제목**: 매력적이고 키워드를 포함하는가?
- [ ] **도입부**: 독자의 공감을 얻는가?
- [ ] **인라인 스타일**: 올바르게 적용되었는가?
- [ ] **코드 블록**: 모든 HTML/코드가 감싸졌는가?
